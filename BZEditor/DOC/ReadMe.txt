Сразу надо заметить, что редактор написан на платформе MS .NET Framework 2.0
отсюда есть следствие - работать будет под win 2003, XP и т.д.
при условии установленного MS .NET Framework 2.0 (23M) 
брать тут
[http://www.microsoft.com/downloads/thankyou.aspx?familyId=0856eacb-4362-4b0d-8edd-aab15c5e04f5&displayLang=ru]
или тут
[http://www.mud.ru/files/dotnet/dotnetfx.exe]
для установки MS .NET Framework 2.0 должен быть установлен WindowsInstaller3.0 и выше (2.5M)
у кого-то возможно он уже установлен, вероятно во 2 сервиспаке ХР он уже установлен
брать тут
[http://www.udomlya.ru/~scream/Editor/WindowsInstaller3.1.exe] 
или тут
[http://www.mud.ru/files/dotnet/WindowsInstaller3.1.exe]
(WindowsInstaller3.0 не устанавливается на 98, ибо не поддерживается она больше)

после этого распаквываем архив с редактором и вперед за славой :)
