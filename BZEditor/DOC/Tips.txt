Для изменения масштаба карты можно использовать колесо мыши при нажатой "Ctrl"
Для изменения отображаемого уровня карты можно использовать колесо мыши при нажатой "Shift"
Для прокрутки карты по горизонтали можно использовать колесо мыши при нажатой "Alt"
Для прокрутки карты по горизонтали и вертикали может использоваться мышь при нажатых "Shift" и правой клавише мыши
Для отмены изменений, сделанных в зоне, достойного механизма отката нет :( но можно воспользоваться первой слева кнопкой тулбара "Перечитать зону из файла без сохранения". Это приведет к откату на последнее сохраненное состояние зоны.
Кнопкой тулбара "Удалить" удаляются как выбранные в списке комнаты, так и комнаты, выделенные на карте
